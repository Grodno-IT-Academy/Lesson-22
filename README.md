Обновляем наш `pip`
```shell
pip install --upgrade pip
```
Установим `jupyter notebook` в нашей виртуальной среде!
```shell
pipenv install jupyter
```
```shell
pipenv install ipykernel
pipenv shell
```
```shell
python -m ipykernel install --user --name=my-virtualenv-name
```
```shell
jupyter notebook
```
